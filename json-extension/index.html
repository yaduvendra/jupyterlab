@jupyterlab/json-extension

json-extension

A JupyterLab extension for rendering JSON as a tree

Prerequisites

  • JupyterLab ^0.27.0

Usage

To render JSON-able dict or list in IPython:

from IPython.display import JSON

JSON({
    "string": "string",
    "array": [1, 2, 3],
    "bool": True,
    "object": {
        "foo": "bar"
    }
})

To render a fully expanded tree:

JSON({
    "string": "string",
    "array": [1, 2, 3],
    "bool": True,
    "object": {
        "foo": "bar"
    }
}, expanded=True)
